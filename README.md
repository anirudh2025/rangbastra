# RANGBASTRA.LUXURY

> Luxury Couture Website built with the Cloutora Luxury Framework.

---

## Overview

Rangbastra is a premium couture website designed to deliver a cinematic, editorial and timeless digital experience.

The project is built using the Cloutora Luxury Framework with a documentation-first development philosophy.

This repository prioritizes:

- Luxury UX
- Scalable Architecture
- Modular Components
- Design Systems
- Documentation
- Performance
- Accessibility
- AI-Assisted Development

# Working Principles

Always:

- Inspect before modifying.
- Reuse before creating.
- Protect the architecture.
- Protect the design system.
- Protect documentation.
- Optimize for long-term maintainability.

Never:

- Introduce unnecessary technical debt.
- Hardcode values that already exist as tokens.
- Replace existing systems without justification.
- Ignore documentation.
