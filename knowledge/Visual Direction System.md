📖 Rangbastra Visual Direction System
Version 1.0
"Designing Emotions, Not Pages."
1. Core Philosophy

Rangbastra is not a website.

It is a luxury couture experience.

Visitors should never feel they are navigating pages.

They should feel they are walking through different rooms of a couture maison.

Every page must possess its own emotional identity.

If two pages feel visually similar, one of them is wrong.

2. The Rule of Distinction

Every experience must answer:

"Why couldn't this section exist on another page?"

If the answer is unclear,

redesign it.

3. Every Page Has A Personality

Instead of designing pages,

design personalities.

Homepage

Emotion:

Invitation.

Feel like:

A luxury film trailer.

Collections

Emotion:

Discovery.

Feel like:

Walking through an art exhibition.

Product Experience

Emotion:

Ownership.

Feel like:

Opening an heirloom jewellery box.

Design Your Outfit

Emotion:

Conversation.

Feel like:

Meeting your personal designer.

Our Story

Emotion:

Trust.

Feel like:

Reading the biography of an artist.

Lookbook

Emotion:

Inspiration.

Feel like:

Browsing Vogue India.

Never feel like scrolling Instagram.

Bride Stories

Emotion:

Memory.

Feel like:

Reading handwritten letters.

Journal

Emotion:

Knowledge.

Feel like:

Luxury editorial publishing.

4. Layout Diversity System

This becomes one of the most important rules.

Never repeat section composition.

Create reusable layout archetypes.

Layout A

Massive photography.

Tiny typography.

Layout B

Huge typography.

No image.

Only whitespace.

Layout C

Editorial collage.

Magazine composition.

Broken alignment.

Controlled asymmetry.

Layout D

Timeline storytelling.

Layout E

Floating content.

Cards are optional.

Layout F

Full-screen statement.

One message.

Nothing else.

Layout G

Split editorial.

Photography dominates.

Copy follows.

Layout H

Immersive horizontal storytelling.

A page should combine different archetypes.

Never:

Heading

↓

Paragraph

↓

Image

↓

Cards

↓

CTA

Repeated forever.

5. The Rule of Surprise

Every third section must surprise the visitor.

Examples:

Unexpected typography.

Unexpected composition.

Unexpected interaction.

Unexpected motion.

Unexpected silence.

Never predictability.

6. Cards Are Not Mandatory

Cards exist only when they improve comprehension.

Never create cards simply because other pages use cards.

7. White Space Philosophy

White space is luxury.

Empty space is laziness.

Every large gap must create anticipation.

If it creates confusion,

remove it.

8. Photography Hierarchy

Photography is the hero.

Text supports photography.

Never the opposite.

Some sections should let images speak entirely on their own.

9. Typography Hierarchy

Typography should occasionally become the hero.

Some pages should tell stories using words alone.

Do not rely on images everywhere.

10. Motion Philosophy

Animation never decorates.

Animation guides.

Every animation must have purpose.

Luxury moves quietly.

11. CTA Philosophy

CTAs should never feel like software buttons.

They should feel like invitations.

Every CTA should communicate:

Elegance.

Confidence.

Exclusivity.

12. Visual Rhythm

The visitor should never predict the next section.

Rhythm:

Dense.

Quiet.

Editorial.

Immersive.

Breathing.

Then surprise.

13. Signature Moments

Every page must contain one unforgettable moment.

Examples:

A cinematic photograph.

A typography composition.

A luxury animation.

An immersive transition.

An emotional quote.

If a visitor screenshots one thing,

that should be it.

14. Icon Philosophy

Icons explain.

Never decorate.

Every icon belongs to the same family.

Never mix styles.

15. Color Philosophy

Black creates focus.

Brown creates warmth.

Gold creates celebration.

Gradient creates emphasis.

Nothing else competes.

16. Section Evaluation Test

Before approving any section, ask:

Does this section deserve to exist?
Does it continue the emotional journey?
Is it visually different from the previous section?
Would Dior build this the same way?
Would Vogue publish this composition?
Does it feel rare?
Does it feel memorable?
Would someone screenshot it?
Does it create curiosity?
Can I remove 20% without losing meaning?

If any answer is "No",

the section is not finished.

17. The 70–20–10 Rule

Every page should feel like:

70% Familiar enough to navigate confidently.
20% Unexpected enough to feel exciting.
10% Iconic enough to be remembered.
18. The Founder Test

Before shipping any page, ask:

"If someone removed the Rangbastra logo, would I still recognize this as Rangbastra?"

If the answer is No,

the visual identity is not strong enough.

🔒 Permanent Rule

From this moment onward:

No new page may be built until it has its own emotional identity and visual personality.

The website must no longer be "a collection of pages."

It must become a collection of experiences.

👑 One final addition (and I think this is the rule that separates good from unforgettable)

I want to add Rule 19:

The "Screenshot Test"

Every page must contain at least one screenshot-worthy composition.

Not just beautiful.

Something a visitor would genuinely stop and share.

Examples:

A breathtaking typography layout.
A full-screen couture image.
A dramatic editorial quote.
A cinematic transition.
A uniquely composed gallery.
An elegant interactive moment.

If an entire page can be scrolled without creating one "wow, I need to save this" moment, then the page isn't finished.
